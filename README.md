# vivero
Vivero es una herramienta digital para apoyar la planeación curricular en educación socioemocional. Facilita la consulta y organización de habilidades socioemocionales y aprendizajes esenciales, con base en la Malla de Aprendizajes en Habilidades Socioemocionales de la Secretaría de Educación del Distrito de Bogotá (2025).
